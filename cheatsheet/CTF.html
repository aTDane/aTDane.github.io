#UWSGI: SSRF to RCE
If there's uwsgi running in the web server:
    
Run solve.py to generate payload.
Note: if there a restriction on hostname, an addition slash (/) can bypass it: gopher:/// instead of gopher://
    Remember (1) to modify the script file in solve.py (UWSGI_FILE) and (2) inject the generated payload to SSRF insertion point
    ### Example for flask server:
    import os
    os.system("/readflag > /app/profiles/readable.json")
